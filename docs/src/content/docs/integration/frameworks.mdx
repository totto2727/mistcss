## React

```sh
mistcss ./components
# or
mistcss ./components --target=react
```

## Vue

```sh
mistcss ./components --target=vue
```

## Svelte

```sh
mistcss ./components --target=svelte
```

## Astro

```sh
mistcss ./components --target=astro
```

## Hono

```sh
mistcss ./components --target=hono
```

## Multiple targets

```sh
mistcss ./components --target=react --target=vue --target=svelte
```
